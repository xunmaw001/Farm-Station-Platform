# Farm-Station-Platform
这款农场驿站微信小程序，是农场与消费者的便捷桥梁。消费者打开小程序，新鲜直采的农产品一目了然，产地、采摘时间清晰标注，一键下单，配送到家。还能线上认养农作物、家禽，通过监控实时查看生长状况，体验种植养殖乐趣。农场主可便捷发布产品、管理订单，依据销售数据合理规划生产。此外，小程序内农业资讯丰富，为农场主提供实用经验。它打破产销隔阂，带来全新农业消费体验 。
